# Dataset description — DaSilva_2026

This folder contains the datasets used in the article:

*Chronic PM2.5 Exposure and Increased Risk of Hospitalization for Kidney Disease in São Paulo, Brazil*,  
published in *Scientific Reports* (2026).

## Data source

Health data were obtained from **DATASUS**, a Brazilian public health data platform maintained by the Ministry of Health  
(www.datasus.saude.gov.br).

The datasets were downloaded and updated up to the period corresponding to the data analysis and manuscript preparation, completed in **2025**.

## Data format

All datasets are provided in plain text format (`.txt`).

## Dataset description

The files correspond to different groups of kidney diseases, as defined and analyzed in the article:

- **AKI.txt**  
  Acute Kidney Injury (acute kidney diseases).

- **CKD.txt**  
  Chronic Kidney Disease (chronic kidney diseases).

- **GD.txt**  
  Glomerulopathies.

The classification and definitions of these disease groups follow the criteria described in the published article.

## Notes

All datasets are provided in their processed form, as used in the statistical analyses. Detailed information regarding data processing, inclusion criteria, and methodological procedures can be found in the corresponding publication.
